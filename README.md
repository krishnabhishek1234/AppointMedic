# 🏥 AppointMedic  
### Doctor Appointment Booking System  

## 📖 Description  
AppointMedic is a simple and user-friendly web application that allows patients to book doctor appointments online. It reduces waiting time and eliminates manual booking by providing a smooth digital experience.

---

## ✨ Features  
- 🔐 User Registration & Login  
- 🔍 Search Doctors by Specialization  
- 📅 View Available Time Slots  
- ✅ Book & Cancel Appointments  
- 🛠️ Admin Panel for Management  

---

## 🛠️ Tech Stack  
- 💻 Frontend: HTML, CSS, JavaScript  
- ⚙️ Backend: Node.js, Express.js  
- 🗄️ Database: MongoDB  

---

## 🚀 How to Run  

### 1️⃣ Clone the repository , install dependencies then run the server  
```bash
git clone https://github.com/krishnabhishek1234/AppointMedic.git
cd appointmedic
npm install
npm start
```
### Open in browser
```bash
http://localhost:3000
```
